## 如何加入

我们欢迎任何人加入我们，但在加入这个团队之前，有一些条件需要满足:

1. 你必须已经在 Crowdin 上翻译了一部分内容
2. 我们需要你保持一定的活跃，当你活跃程度非常低时，我们会进行匿名投票来决定是否将你从这个团队移除

## 附加
